# DRG Travel & Tours

Travel agency website for my parents' travel business. This is a home-based travel agency mostly run by my mother. It currently has a Facebook page, Tiktok page, and Instagram page except for a website.

## Features

- Home page with company info and travel photos
- Navigation links (About, Services, Gallery)
- Airplane banner image
- Photo gallery with 6 travel destination images
- Contact information section

## Routes

**`/` (Home Page)**
- Main landing page
- Welcome message with airplane image
- "Who We Are", "What We Do", "How We Work" sections
- Travel photo gallery
- Contact information and inquiry section

**`/about` (About Page)**
- Navigation link ready
- Not yet implemented

**`/services` (Services Page)**
- Navigation link ready
- Not yet implemented

**`/gallery` (Gallery Page)**
- Navigation link ready
- Not yet implemented
